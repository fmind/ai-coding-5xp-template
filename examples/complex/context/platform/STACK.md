# Technology Stack

## Core Technology Stack

- **Languages**: Python 3.12+ (Backend), TypeScript 5+ (Frontend)
- **Tooling**: uv, pnpm
- **Backend**: Django 5+ (Robust ORM and admin interface)
- **Frontend**: Next.js 14+ (App Router for optimized SEO)
- **Database & Storage**: PostgreSQL 16 (Relational), Redis 7 (Caching & Background Tasks)

## Third-Party Integrations

- **Payments**: Stripe Checkout via custom integrations
- **Authentication**: NextAuth.js linking to internal Postgres DB
- **Search**: Algolia (Catalog indexing)
- **Communication**: Postmark (Transactional emails)
- **Analytics**: Vercel Web Analytics & PostHog

## Tooling & Verification

- **Linting**: Ruff (Python), ESLint (TypeScript)
- **Formatting**: Black (Python), Prettier (TypeScript)
- **CI/CD**: GitHub Actions (Testing & Deployment pipelines)
