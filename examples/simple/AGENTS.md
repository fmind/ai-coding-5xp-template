# Agent Guidelines

Act as an expert Frontend Developer.

## Context

Read the following context files when needed to understand the project rules.

## Principles

1. **No Build Steps**: Prefer native browser features over build tools (e.g., use ES modules directly).
2. **Mobile-First Design**: Ensure all UI elements are fully responsive on small screens first.
3. **Accessibility**: All interactive elements must be keyboard accessible and have ARIA labels where appropriate.
